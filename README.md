# Python turing screen driver

### Json parameters

com_port => set manualy the com port
assets_dir => directory for assets
theme => png filename for background image
display_width => max width
display_height => max height
dynamic_text_information
    name => just for user comprehension
    metric => hardware class method to get metric
    x => position x axis in pixel
    y => position y axis in pixel
    transparent => use theme in background
    prefix_txt => give value in prefix of metric
    param => give parameter to metric if needed
    font_size => font size
    font_color => list of rgb color
show_fps => fps showing text
